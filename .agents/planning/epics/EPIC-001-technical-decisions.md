# EPIC-001: Decisões Técnicas Fundamentais (RFCs)

**Status:** `[IN_PROGRESS]`
**Responsável:** Equipe Técnica / CTO / Agentes Especializados

## Descrição
Este épico agrega as decisões críticas de longo prazo necessárias para estabilizar a evolução da arquitetura do App do Artesão. Cada decisão listada aqui possivelmente gerará um Request for Comments (RFC) criado de forma colaborativa com Agentes IA especializados.

## Objetivos
- Garantir a resiliência dos dados do usuário (Offline-first, Migrations).
- Estabelecer pipeline automatizado (Local -> GitHub Pages).
- Definir estratégia de Testes de Integração e E2E.
- Elaborar design seguro de Sync em nuvem sem backend dedicado.
